# Store API iOS App

A modern iOS application built with SwiftUI that displays store items in a clean and efficient interface.

## Features

- Asynchronous image loading for store items
- Clean and modern UI design
- Responsive layout with proper scaling
- Category-based item organization
- Loading indicators for better user experience

## Technical Details

### Requirements

- iOS 15.0+
- Xcode 13.0+
- Swift 5.5+

### Architecture

The app is built using SwiftUI, Apple's modern declarative UI framework, which provides:
- Declarative syntax for building user interfaces
- Automatic layout and animation
- Native performance optimization

### Key Components

#### StoreRowView
A reusable SwiftUI view component that displays individual store items with:
- Asynchronous image loading
- Item title and category information
- Proper image scaling and fitting
- Loading indicator for image loading states
